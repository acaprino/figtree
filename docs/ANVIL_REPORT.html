CSS Architecture

Key principle: All opacity variants use color-mix(in srgb, var(--token) N%, transparent) -- zero hardcoded rgba values. This ensures every color automatically adapts when themes change.

Transitions: 150ms ease-out on structural containers for smooth theme crossfade. Retro mode disables all transitions.

Border radius: --radius-sm (4px), --radius-md (6px). Retro mode sets both to 0px.

Active states: transform: scale(0.97) pressed effect on buttons.

Focus states: outline: 2px solid var(--accent); outline-offset: 2px.

Psychology & Engagement

Power User Identity

40+ keyboard shortcuts, retro/hacker themes (Matrix, Cyberpunk), terminal-first view mode, zero onboarding -- signals "this app is for expert developers".

Investment & Lock-in

Session persistence (resume/fork), custom system prompts, project labels, per-project usage stats -- users accumulate personalization that increases switching cost.

Transparency & Trust

Real-time cost display ($0.XXX per session), context window meter, rate limit indicator, full token breakdowns -- builds trust through radical cost visibility.

Autonomy Spectrum

Three permission modes (plan / accept edits / skip all) + security gate toggle -- gives users precise control over AI autonomy level.

SWOT Analysis

StrengthsWeaknesses
  • Native Tauri 2 performance (not Electron)
  • Keyboard-first UX (40+ shortcuts)
  • 14 themes including unique retro aesthetics
  • Dual view: Terminal + Chat
  • Session resume/fork -- zero work loss
  • Real-time cost & context tracking
  • Plugin marketplace via skills
  • Windows-only (no macOS/Linux)
  • No onboarding -- steep learning curve
  • No automated tests
  • Single AI provider (Claude only)
  • Session transcript viewer is TODO
  • No collaborative features
OpportunitiesThreats
  • Cross-platform (Tauri 2 supports it)
  • Multi-provider support
  • Session sharing/export
  • Inline diff viewer for code changes
  • Git-aware session management
  • Project templates
  • Claude Code CLI improving natively
  • VS Code AI extensions growing
  • Cursor and AI-native IDEs
  • Possible API pricing changes
